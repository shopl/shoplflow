# shoplflow에 디자인 토큰 반영하기

<Steps>
    ### 왼쪽 하단에 Push to Github 누르기

    ![](/push-shoplflow/step1.png)

    ### 빨간색 영역에 shopl/shoplflow 으로 설정되어있는지 확인해주세요.
    만약 설정되어있지 않다면 다시 설정해주세요.

    - Commit message

    어떤 변경 사항이 있는지에 대한 메모를 작성하는 곳이에요.
    간단하게 요약해서 작성해주세요.
    - Branch

    feature/token 으로 설정되어있는지 확인해주세요.

    ![](/push-shoplflow/step2.png)

    ### Push Changes를 눌러주세요. 이후 create Pull Request를 눌러주세요.

    ![](/push-shoplflow/step3.png)

    만약 제목으로 업데이트가 설명이 가능하다면 create pull request를 눌러주세요.

    만약 제목만으로 업데이트를 설명할 수 없다면 아래 설명을 작성하고 create pull request를 눌러주세요.

    ### 왼쪽 하단의 Labels를 눌러 `token-update`라는 라벨을 선택해주세요.

    선택해주셔야 코드를 자동 생성해줘요.

    ![](/push-shoplflow/step4.png)

    ### 최종 확인해주세요.
    - 제목이 입력되었나요?
    - **Reviewers**에 리뷰어가 한 명 이상 할당되어있나요?
    - **Labels에 `token-update`** 라는 라벨이 추가되어있나요?

    ![](/push-shoplflow/step5.png)

    ### 잘 생성되었다면 아래와 같은 댓글이 추가돼요.

    표시된 부분을 눌러주세요.

    ![](/push-shoplflow/step6.png)


    ### 빨간색 부분에 다음과 같이 적혀있는지 확인해주세요.
    (다르다면 개발자를 소환해주세요.)

    ![](/push-shoplflow/step7.png)

    **녹색 영역에는 자세한 업데이트 내역을 작성해주세요.**
    (변경사항 로그 문서에 추가되는 내용입니다.)

    **위 과정을 완료했다면 Commit Changes를 눌러주세요!**

    **이후에 나타나는 모달도 확인 눌러주시면 됩니다.**

    ### 화면과 같이 표시됐다면 성공입니다!

    ![](/push-shoplflow/step8.png)



</Steps>
