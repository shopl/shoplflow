# Usage: Build and Deploy

패키지를 빌드하고 배포하는 방법을 알아봅시다.

## Build

이 라이브러리는 `esbuild`를 기반으로 하는 `tsup`을 사용하여 빌드됩니다. `pnpm build`을 실행하면 전체 프로젝트가 빌드돼요.

## Deploy

Github Action을 기반으로 Storybook을 자동배포해요.

<Callout>
 수동 배포 로직도 추가할 예정이에요.
</Callout>

## Release

이 라이브러리는 `npm`에 배포되지 않고 버전관리만 진행하고 있어요.

📌 신규 기능이 추가되었다면 `yarn workspace @hada/ui-kit release:minor`를 실행해요.

📌 버그 수정이나 기능 개선이라면 `yarn workspace @hada/ui-kit release:patch`를 실행해요.


```shell
yarn workspace @hada/ui-kit release

yarn workspace @hada/ui-kit release:patch

yarn workspace @hada/ui-kit release:minor
```
