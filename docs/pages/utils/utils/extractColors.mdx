# extractColors

문자+숫자로 이루어진 키를 가진 객체에서 특정 문자로 시작하는 키를 가진 객체를 추출합니다.

## Usage
```ts
import {extractColors} from "@shoplflow/utils"
const primaryColors = extractColors(tokens, 'primary');
```
