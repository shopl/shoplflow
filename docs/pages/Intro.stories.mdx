# Hada Design System

하다 프로젝트의 디자인 시스템 라이브러리입니다.

> 임시적으로 `@emotion`과 `@mui`를 같이 사용하고 있으나 점진적으로 `@mui`를 제거할 예정이에요.
>
> 이 문서에 노출되는 컴포넌트는 모두 `@emotion`으로만 개발된 컴포넌트에요.
>
> **이 라이브러리에서 사용하는 단위는 `rem`을 사용하지만 문서 상의 편의를 위해 px로 표기하고 있어요.**

## 프로젝트 세팅 방법

- @emotion의 타입추론을 위해 사용할 프로젝트 `src/` 경로에 `emotion.d.ts` 파일을 생성한 후 하단의 코드를 넣어주세요.

```ts
import '@emotion/react';
import { ThemeType } from '@hada/ui-kit';

declare module '@emotion/react' {
  export interface Theme extends ThemeType {}
}
```

- Theme을 적용하기 위해 App.tsx나 index.tsx에 아래 코드를 추가해주세요.

```tsx
import {HdsProvider} from '@hada/ui-kit';
const App = () => {
  return (
    <HdsProvider>
        <YourComponent/>
    </HdsProvider>
  );
};
```

- normalizeCss를 적용하려면 HdsProvider에 `normalizeCss` prop을 추가해주세요.

```tsx
import {HdsProvider} from '@hada/ui-kit';
const App = () => {
  return (
    <HdsProvider normalizeCss>
        <YourComponent/>
    </HdsProvider>
  );
};
```
