# Dropdown

Dropdown은 총 세가지 컴포넌트로 나뉘어집니다.

- Dropdown
- DropdownButton
- DropdownContent

> Dropdown option
> - CLICK: 화면을 클릭하면 닫힙니다.
> - OUTSIDE_CLICK: 컨텐츠 영역 외부를 클릭하면 닫힙니다.
> - NONE: 클릭으로 닫히지 않습니다.

## Usage

```
<Dropdown
  isOpen={true}
  trigger={
    <InputButton
      placeholder={'Dropdown 안에 InputButton을 넣었어요.'}
      rightSource={<IconButton iconSource={RightArrowIcon} styleVar={'GHOST'} sizeVar={'S'} />}
    />
  }
  content={<Dropdown.Content type={'FILL'}>Content</Dropdown.Content>}
/>
```
