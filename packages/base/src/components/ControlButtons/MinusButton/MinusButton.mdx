# Minus button
Minus button은 요소를 삭제할 때 사용되는 컴포넌트입니다.

> - default 색상이 적용되어 있으나, 필요할 경우 `color` 속성으로 배경 색상을 변경할 수 있습니다. 
> - hover할 경우 배경 색상은 `color` 보다 한 단계 높은 colorToken으로 적용됩니다.
