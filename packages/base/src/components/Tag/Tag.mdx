# Tag

Tag는 특정 정보를 보여줄 때 사용하는 데이터 표시 컴포넌트입니다.

color를 사용하면 styleVar에 따라 스타일이 변경됩니다. 각 스타일은 채도가 높은 색상을 기준으로 변경됩니다.

## Usage

```tsx
const App = () => {

  return (
    <Tag color={'red300'} sizeVar={'M'}>
      태그에요
    </Tag>
  );
};
```
