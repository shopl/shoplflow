# Minus button

Minus button은 일괄 등록/수정 테이블에만 사용되는 컴포넌트입니다.

클릭시 selected 상태로 전환되는 Checkbox 컴포넌트와 달리 `onClick` 속성으로 전달된 함수를 실행합니다.

default 색상이 적용되어 있으나, 필요할 경우 `color`, `hoverColor` 속성으로 배경 색상을 변경할 수 있습니다.

<Canvas of={MinusBoxStories.Default} />

## Usage


```ts filename="MinusButton"
// default
import { MinusButton } from '@shoplflow/base';

const Example = () => {

    return (
        <MinusButton onChange={() => { console.log('예시입니다') }}/>
    )
}
```

```ts filename="MinusButton"
// color change
import { MinusButton } from '@shoplflow/base';

const Example = () => {

    return (
        <MinusButton color={'shopl300'} hoverColor={'shopl400'} onChange={() => { console.log('예시입니다') }}/>
    )
}
```
