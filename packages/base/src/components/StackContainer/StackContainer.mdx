# StackContainer & MotionStack

> 스토리북 내부에서는 iframe으로 동작하기 때문에 정상적으로 보이지 않을 수 있습니다.
> 오른쪽 상단의 `Open canvas in new tab`을 통해 확인할 수 있습니다.

StackContainer 컴포넌트는 자식 컴포넌트의 레이아웃 영역을 감싸는 용도로 사용해요.

Stack 컴포넌트와 동일하게 사용할 수 있지만 Container의 목적을 가지고 있기 때문에 padding, margin, background, radius를 사용할 수 있어요.

`as` 키워드로 HTML 태그를 변경할 수 있어요. (기본값: `div`)

`framer-motion`을 사용한 `MotionStack`도 제공해요. 사용방법은 `framer-motion`의 컴포넌트와 동일해요. `animate`와 `initial`속성을 통해 애니메이션을 적용할 수 있어요.

`StackContainer.Horizontal`과 `StackContainer.Vertical`을 통해 각각 수평 정렬과 수직 정렬이 기본값인 Stack을 사용할 수 있어요.
