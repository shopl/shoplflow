# List
리스트는 특정 데이터를 나열할 때 사용하며 2가지 컴포넌트로 구성되어 있습니다.

- List
- List.Text2Rows


<img src={AnatomyList} alt={'anatomy-list'}/>

<br/>
<br/>

#### 1. leftSource 체크박스, 라디오버튼 등의 왼쪽에 위치하는 컴포넌트를 표현합니다.

#### 2. children 리스트 내부에 들어가는 컴포넌트를 표현합니다. Text2Rows나 Avatar 컴포넌트가 들어갈 수 있습니다.

#### 3. rightSource 리스트 오른쪽에 위치하는 컴포넌트를 표현합니다.

<Canvas of={ListStories.Playground} />


## Usage
```tsx
import React from 'react';

const App = () => {
  return (
    <List {...args} rightSource={<Text whiteSpace={'nowrap'}>서브 데이터</Text>}>
      <StackContainer height={'32px'} width={'32px'} background={'neutral300'} radius={'borderRadius16'} />
      <List.Text2Rows title={'타이틀'} subTitle={'서브 타이틀'} />
    </List>
  )
}
```
