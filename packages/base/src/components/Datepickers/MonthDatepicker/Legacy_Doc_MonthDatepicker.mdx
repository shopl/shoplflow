# MonthDatepicker
월을 선택하는 데이트피커에요.
> - 시작월과 마지막월을 시작하면 해당 기간 안에 존재하는 모든 날짜를 호출하는 컴포넌트에서 받을 수 있어요.
> - onlySingleMonth 프롭을 전달하면 한달만 선택 가능해요.

## Usage
```tsx
import { MonthDatepicker} from '@shoplflow/base'
import type { MonthClickDateInfo } from '@shopflow/base';

const Example = () => {
  const [selectedDateInfo, setSelectedDateInfo] = useState<MonthClickDateInfo | null>(null);

  return (
    <MonthDatepicker
        onlySingleMonth
        handleMonthClick={(dataInfo) => {
          setSelectedDateInfo(dataInfo);
        }}
      />
    );
}
```
