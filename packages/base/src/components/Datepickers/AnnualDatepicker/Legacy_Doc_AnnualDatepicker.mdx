# AnnualDatepicker
- 연도를 선택할 수 있는 데이터피커에요.
- 연도는 단일 선택만 가능한 단순한 컴포넌트에요.

<Canvas of={AnnualDatepickerStories.Playground} />

## Usage
```tsx
import { AnnualDatepickerStories } from './AnnualDatepicker;

const Example = () => {
  return (
      <AnnualDatepicker
        currentYear={selectedYear}
        disabledYears={[2020, 2030, 2040]}
        handleYearClick={(year) => {
          setSelectedYear(year);
        }}
      />
  );
};
```
