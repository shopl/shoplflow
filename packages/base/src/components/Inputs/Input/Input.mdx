# Input
html의 `input`과 동일하게 사용할 수 있어요.

> - 최대 글자수가 지정될 경우에만 포커싱할 때 글자수 카운터가 표시돼요.
> - 만약 글자가 입력된 경우 우측에 X 버튼이 노출되며 클릭 시 입력된 글자가 모두 삭제돼요.

<Canvas of={InputStories.Playground} />

## Usage
```tsx
import { StackContainer, Input } from "@shoplflow/base";

const Example = () => {

  const inputRef = useRef<HTMLInputElement>(null);

  return (
    <StackContainer width={'500px'} height={'300px'} justify={'center'}>
      <Input maxLength={50} ref={inputRef} />
    </StackContainer>
  );
}
```
