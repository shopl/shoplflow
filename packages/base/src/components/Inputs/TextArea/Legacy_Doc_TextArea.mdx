# TextArea
html의 `textarea`와 동일하게 사용할 수 있으며, 기본적으로 `resize: none` 속성이 적용되어 있습니다.

> - `leftSource`, `rightSource` 속성을 사용해 하단의 Toolbar를 설정할 수 있습니다. 기본적으로는 우측 하단에 `<TextArea.MaxLength/>`가 노출됩니다.
> - Toolbar에 글자수/최대 글자수를 표시하고 싶다면 `<TextArea.MaxLength/>`를 사용해주세요. 이 때 최대 글자 수 속성인 `maxLength` 이 설정되어 있어야 노출됩니다.
> - TextArea가 disabled 상태일 경우 Toolbar의 영역은 유지되지만 요소는 보이지 않습니다.

## Usage
```tsx
import { TextArea } from '@shoplflow/base';


const App = () => {
    return (
      <TextArea />
    )
}
```

```tsx
// with maxLength
import { TextArea } from '@shoplflow/base';


const App = () => {
    return (
        <TextArea maxLength={30} />
    )
}
```
```tsx
// Disabled
import { TextArea } from '@shoplflow/base';


const App = () => {
    return (
        <TextArea maxLength={30} disabled />
    )
}
```

```tsx
// with custom Toolbar
import { TextArea } from '@shoplflow/base';


const App = () => {
    return (
        <TextArea maxLength={30} leftSource={<TextArea.MaxLength/>} rightSource={<Button sizeVar='S'>확인</Button>} />
    )
}
```
