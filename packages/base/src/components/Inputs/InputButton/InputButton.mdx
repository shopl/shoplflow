# InputButton

`InputButton` 컴포넌트는 값을 선택하는 로직과 상호작용할 수 있는 버튼 컴포넌트입니다.

값이 추가/변경되거나 삭제될 때마다 `onChange` 콜백이 호출됩니다.

값이 존재할 경우 X 버튼이 노출되며 클릭시 onClear 콜백이 호출됩니다.
