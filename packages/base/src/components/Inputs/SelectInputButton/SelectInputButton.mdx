# SelectInputButton
SelectInputButton은 InputButton에 Multiple Select 기능을 추가한 컴포넌트입니다.
> - 선택한 최초 값만 표시되며 최초 값 외의 값은 +n으로 표시됩니다.

## Overview
<Canvas of={SelectInputButtonStories.Playground} />

## Usage
```tsx
import SelectInputButton from './SelectInputButton';

export default function Example() {
  const options = [
    { id: 1, name: 'Option 1' },
    { id: 2, name: 'Option 2' },
    { id: 3, name: 'Option 3' },
  ];

  return (
    <SelectInputButton
      value={options}
      label="name"
      placeholder="옵션을 선택하세요"
      onClick={() => console.log('클릭')}
      onClear={() => console.log('클리어')}
      rightSource={<span style={{ color: 'blue' }}>더보기</span>}
      disabled={false}
    />
  );
}
```
