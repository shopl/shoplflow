# Skeleton

Skeleton은 데이터가 로드되는 동안 사용자에게 로딩 상태를 시각적으로 표시하는 컴포넌트입니다. Skeleton 컴포넌트는 재사용 가능한 기본 스켈레톤 스타일을 제공하여 다른 스켈레톤 조합에 활용할 수 있습니다.

## Usage

Skeleton 컴포넌트는 다음과 같이 사용할 수 있습니다:

```tsx
import Skeleton from './Skeleton';

function LoadingState() {
  return (
    <div>
      <Skeleton styleVar={'circle'} width="200px" />
      <Skeleton styleVar={'circle'} width="150px" />
      <Skeleton styleVar={'rectangle'} width="100px" height="20px" />
    </div>
  );
}
```
