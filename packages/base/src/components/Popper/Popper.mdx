# Popper

특정 요소를 기준으로 pop up을 띄워주는 컴포넌트입니다.

로직만 분기되어있으며 디자인은 포함되어 있지 않습니다.

Dropdown, Tooltip 등으로 활용할 수 있습니다.

> [floating-ui](https://floating-ui.com)를 기반으로 작성되었습니다.

## Usage

```
import { Popper } from '@shoplflow/base';

const App = () => {
  return(
    <Popper>
      <Popper.Trigger isOpen={isOpen}>
        <Button onClick={handleToggle}>Popper Playground</Button>
      </Popper.Trigger>
      <Popper.Portal>
        <Modal.Container>
          <Modal.Body>팝오버 안에 모달 넣기</Modal.Body>
        </Modal.Container>
      </Popper.Portal>
    </Popper>
  )
}
```
