# Button
가장 기본적인 형태의 버튼입니다.

> - `hover`상태의 배경색, `border-color`는 배경색의 상위 colorToken으로 자동 설정됩니다.
> - 버튼의 배경색은 기본적으로 `styleVar`에 따라 지정되어 있습니다. 커스텀하고 싶을 경우 `styleVar`를 `SOLID`로 지정하고, `color`값을 지정해주세요.

## Usage
```tsx
import { Button } from '@shoplflow/base';

const Example = () => {
  return(
    <Button styleVar='PRIMARY' sizeVar='M'>원하는 내용을 입력하세요</Button>
    );
};
```
