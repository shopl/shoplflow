# Button
디자인 시스템에서 사용하는 버튼 컴포넌트에요. 

> - styleVar 중 SOLID 속성만 메인 color를 변경할 수 있습니다.
> - as 속성을 이용해 HTML 태그를 변경할 수 있어요. (기본값: button)

## Overview
<Canvas of={ButtonStories.Primary} />

## Usage
```tsx
import { Button } from '@shoplflow/base';

const Example = () => {
  return <Button styleVar='PRIMARY' buttonSize='M'>원하는 내용을 입력하세요</Button>;
};
```
