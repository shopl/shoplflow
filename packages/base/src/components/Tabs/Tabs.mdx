# Tabs

단일 Tab으로  SwitchCase의 노출 View를 변동하는 컴포넌트입니다.

- 탭을 컨트롤하기 위해 onChange: tab(string) => void 시그니처의 함수를 Tabs에 Props으로 넘겨주세요

## Usage

```typescript
import { Tabs } from '@shoplflow/base';

const App = () => {

  const [activeTab, setActiveTab] = useState('value1');

  return (
    <>
      <Tabs initialTab={activeTab} onChange={setActiveTab}>
        <Stack.Horizontal>
          <Tabs.Tab value={'value1'} label={'label1'} />
          <Tabs.Tab value={'value2'} label={'label2'} />
          <Tabs.Tab value={'value3'} label={'label3'} />
        </Stack.Horizontal>
      </Tabs>
      {activeTab === "a" && <div>acitve a</div>}
      {activeTab === "b" && <div>acitve a</div>}
      {activeTab === "c" && <div>acitve a</div>}
    </>
  )
}
```
