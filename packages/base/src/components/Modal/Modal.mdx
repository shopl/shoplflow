# Modal

Modal 컴포넌트는 Window 위에 추가적으로 나타나는 팝업입니다.

Modal 컴포넌트는 4가지 컴포넌트로 구성되어있습니다.

- Modal.Container (Modal 컴포넌트의 최상위 컴포넌트)
- Modal.Header (Modal 컴포넌트의 헤더 컴포넌트)
- Modal.Body (Modal 컴포넌트의 바디 컴포넌트)
- Modal.Footer (Modal 컴포넌트의 푸터 컴포넌트)
