# Modal

> **기본 스크롤바로 인한 layout shift 현상을 개선하기 위해 스크롤바 컴포넌트를 사용하고 있습니다.**

Modal 컴포넌트는 Window 위에 추가적으로 나타나는 팝업입니다.

Modal 컴포넌트는 4가지 컴포넌트로 구성되어있습니다.

- Modal.Container (Modal 컴포넌트의 최상위 컴포넌트)
- Modal.Header (Modal 컴포넌트의 헤더 컴포넌트)
- Modal.Body (Modal 컴포넌트의 바디 컴포넌트)
- Modal.Footer (Modal 컴포넌트의 푸터 컴포넌트)

> **아래의 조건으로 스타일이 변경됩니다.**
- Header가 있는 경우: Body 상단 padding 16px
- Header가 없는 경우: Body 상단 padding 24px

## Usage
ShoplflowProvider에는 모달을 열고 닫는 Portal이 포함되어있습니다.

모달을 열고 닫는 handle 함수는 **`useHandleModal`** hook을 통해 제공됩니다.

모달 상태를 확인하는 **`useModalValue`** hook을 통해 모달 상태를 확인할 수 있습니다.
