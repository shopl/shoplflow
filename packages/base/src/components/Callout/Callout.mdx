# Callout

Callout은 중요한 알림이나 정보를 보여주는 메세지를 나타낼 때 사용합니다.

## Usage

```tsx
import React from 'react';
import { StackContainer, Callout } from '@shoplflow/base';

export const App = () => {
  return (
    <StackContainer width={'400px'}>
      <Callout styleVar={'INFORMATION'}>
        <Callout.Icon iconSource={NoticeIcon} />
        <Callout.Text>
          사용 안 함으로 설정할 경우 기존 등록된 게시물을 읽을 수도, 새로운 게시물을 포스팅할 수도 없게 됩니다. (단,
          기존 등록된 게시물이 삭제되지는 않습니다.)
        </Callout.Text>
      </Callout>
    </StackContainer>
  );
};
```
